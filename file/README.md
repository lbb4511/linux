# Linux 文件系统

- `mkdir` 新建文件夹
- `touch` 新建文本文件

- `ls` 查看文件目录
- `ls -a` 查看全部文件
- `ls -l` 查看文件详细信息

- `cp` 复制文件
- `cp -r` 递归复制整个文件夹
- `cp -v` 显示详细复制信息
- `mv` 移动文件，有重命名作用参数同 cp
- `rm` 删除文件
- `rm -i` 交互式删除，有提示
- `rm -r` 删除文件夹
- `rm -f` 强制删除
- `rmdir` 删除空文件夹
